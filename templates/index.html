StockVision AI

StockVision AI is an ML-powered stock prediction tool that forecasts two-day prices for symbols like AAPL using Prophet, XGBoost, and LightGBM, offering intuitive visualizations, news sentiment insights, and secure user authentication through a Flask-based web app deployed on Render.

Register

AI Predictions

Powered by Prophet, XGBoost, and LightGBM, StockVision AI forecasts two-day stock prices, incorporating features like lagged prices, moving averages, and a 9% tariff shock adjustment (post-April 2025), with news sentiment summaries enhancing prediction reasoning.

Register

Platform Tools

The platform offers an intuitive dashboard for stock analysis, Matplotlib visualizations (actual prices in blue, predictions in green), and a combined prediction plot averaging outputs from all models, alongside stock info cards for Open, High, Low, Close, and Volume.

Register

User Benefits

StockVision AI provides secure, personalized stock insights with Firebase Authentication, a responsive dark-mode interface (85% user satisfaction), and scalable performance, helping users make data-driven investment decisions.
